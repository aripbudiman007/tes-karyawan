### **Git Rule :**
* Sebelum melakukan push, buatlah `branch` baru dengan nama kalian.
* Commit dengan message `"Tes telah selesai"`.
* Push ke branch masing-masing.
* Tidak diperkenankan untuk push ke branch `master`.
* Buatlah pull request.
